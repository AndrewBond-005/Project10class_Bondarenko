JavaScript


JavaScript — это язык программирования, который используют для написания frontend- и backend-частей сайтов, а также мобильных приложений. В frontend-части сайтов язык используют для создания интерактива (анимаций, всплывающих форм, автозаполнения), так как он связан с HTML и CSS и может ими манипулировать, а в backend-части с языком JavaScript работают на платформе Node.js и с ее помощью, например, разрабатывают серверные веб-приложения и подключают библиотеки. Часто в текстах и обучающих материалах название языка сокращают до JS. Это язык программирования высокого уровня, то есть код на нем понятный и хорошо читается.
