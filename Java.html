Java


Java — объектно-ориентированный язык программирования, разрабатываемый компанией Sun Microsystems и официально выпущенный 23 мая 1995 года, и был разработан Джеймсом Гослингом. В отношении произношения в русском языке, как и в ряде других, образовались две различные нормы — заимствованная англоязычная /ˈdʒɑːvə/ и традиционно-национальная «Ява», соответствующая традиционому произношению названия острова Ява.
